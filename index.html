Om servern

SeksisSMP är en vänlig och aktiv Minecraft SMP-server där du kan spela, bygga och ha kul med andra spelare. Vi har en trevlig community och regelbundna event!

Ansök här för att gå med!
Gå med i Discord

Hur går jag med?

  1. Du måste ansöka för att bli whitelistad!
  2. Starta Minecraft (Java Edition rekommenderas).
  3. Gå till "Multiplayer" och klicka på "Add Server".
  4. Skriv in serverns IP och anslut!

Regler

  • Visa respekt mot andra spelare.
  • Inget fusk eller hacks.
  • Ingen griefing eller stöld.
  • Ha kul!

Vanliga frågor

Hur lång tid tar det att bli whitelistad?
Oftast inom 24 timmar. Håll utkik på Discord!
Får jag bjuda in kompisar?
Ja, men de måste också ansöka och bli godkända.
Hur använder jag proximity chat (röstchatt) på servern?
Steg 1: Installera Forge eller Fabric
Forge: https://files.minecraftforge.net/
Fabric: https://fabricmc.net/use/ + Fabric API
Installera rätt version för Minecraft.

Steg 2: Ladda ner Simple Voice Chat
https://modrinth.com/mod/simple-voice-chat
Ladda ner .jar som matchar Forge/Fabric + rätt Minecraft-version.

Steg 3: Lägg i mods-mappen
1. Win + R → %appdata% → .minecraft
2. Öppna eller skapa "mods"-mapp
3. Lägg in .jar-filen (och Fabric API om du kör Fabric)

Steg 4: Starta Minecraft
1. Välj Forge/Fabric-profil i launchern
2. Starta spelet – ser du mikrofon-ikon? Klart!

Steg 5: Anslut till Seskis SMP
1. Gå in på servern
2. Tryck V → välj mic, ljud, volym
3. Prata nära andra spelare
